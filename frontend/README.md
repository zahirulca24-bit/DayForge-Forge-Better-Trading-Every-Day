# DrayFrogd Frontend

React frontend for the FastAPI-powered intraday auto-trading bot.

## Local Run

1. Install dependencies:
   `npm install`
2. Set `VITE_API_BASE_URL` in `.env.local` if your backend is not on `http://127.0.0.1:8000`
3. Start the dev server:
   `npm run dev`
4. Build for production:
   `npm run build`
